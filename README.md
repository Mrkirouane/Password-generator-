# Password Generator

Simple client-side password generator built with HTML, CSS and JavaScript.

## Files
- `index.html` — main page
- `style.css` — styles (responsive)
- `index.js` — password generation logic

## Run locally
Open `index.html` in your browser. For a local server (optional):

PowerShell:
```
python -m http.server 8000
```

Then open `http://localhost:8000` in your browser.
